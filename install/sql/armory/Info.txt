MaNGOS Blizzlike Armory (MBA)

Blizzlike armory for people using MaNGOS.
The Armory is written in PHP script. Works with PHP5.
Now it supports 2.4.3, 3.0.9, 3.1.3 and 3.2.2.
Supported languages: English, German, Russian and French.
